Podcast Player Pro adds extra player display styles for list and grid presentation.

## Available Pro Styles

- `lv1`: List - Normal
- `lv2`: List - Small
- `lv3`: List - Minimal
- `lv4`: List - Individual
- `gv1`: Grid - Normal
- `gv2`: Grid - Cards

 The free styles also remain available, including the default, modern, and legacy/catalogue styles.

## Steps

1. Edit the player in the shortcode builder, block, or widget.
2. Open **Podcast Player Display Style**.
3. Choose the layout.
4. For grid styles, set **Maximum Grid Columns**.
5. If thumbnails are visible, choose thumbnail cropping options.
6. Save and test on desktop and mobile.

## Manual Shortcode Example

```
[[podcastplayer feed_url="https://example.com/feed.xml" display_style="gv1" grid_columns="3"]]
```

## Notes

- Grid column classes are applied only to styles that support grids.
- Pro list/grid styles support additional background, text, thumbnail, play-button, and excerpt controls depending on the style.
