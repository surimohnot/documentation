Podcast Player Pro can be distributed in two license variants in this codebase: the standard license-key screen and a Freemius-powered premium build. Use the license UI that appears in your WordPress admin.

## Activate a License Key

1. Go to **Podcast Player** in the WordPress admin menu.
2. If the header badge says **Activate Pro**, open the activation area from that notice or from the plugin action link on **Plugins**.
3. Enter the license key.
4. Click **Activate License**.
5. Wait for the confirmation message.
6. Confirm the header status changes to **Pro Active**.

## Deactivate a License Key

1. Go to **Podcast Player**.
2. Open the Pro license section.
3. Click **Deactivate License**.
4. Wait for confirmation before moving the license to another site.

## Freemius Build

If the admin header shows **Pro (Freemius)**, licensing is handled by the Freemius account screens instead of the legacy key form. Use the account, checkout, or license activation links shown by WordPress/Freemius.

## Notes

- A license is not required for the PHP classes to exist, but an inactive or invalid license can block updates and account features.
- If activation fails, verify the key, site URL, activation limit, and network connectivity.
