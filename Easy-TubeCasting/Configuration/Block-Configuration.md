# Block Configuration Settings

When you select an Easy TubeCasting block in the WordPress editor, several customization options appear in the sidebar.

## Primary Settings

- **YouTube URL Field:** Paste a video, playlist, channel, username, or handle URL. You can also paste multiple video URLs separated by commas.
- **Display Style:** The current built-in frontend style is **Spotlight Player**. The editor keeps this value set to `Spotlight Player`.

## Layout Customization

- **Videos Per Batch:** Controls how many playlist items are visible initially and how many additional items appear with each **Load More** click. Default is `10`. This is a display setting; it does not limit how many videos are fetched from YouTube for a collection.
- **Range:** The editor accepts values from `1` to `50`.
- **Single-Player Mode:** Set this value to `1` to hide the playlist and show only the primary player.

## Visual Styling

- **Theme Mode:**
- **Dark:** Sleek, modern dark background (default).
- **Light:** Clean, light background that blends work with standard white page designs.
- **Accent Color:** Controls the play badge and player accent color. If no color is selected, the default is `#0466ff`.

## Pro-Tips for Configuration

- **Responsive Design:** You don't need to worry about mobile devices; the Spotlight player adapts to smaller screens.
- **Large Collections:** If you have a very large channel, keep "Videos Per Batch" at a reasonable number (10-15) so the playlist remains easy to browse.
- **API Key:** A key is only required for playlists, channels, usernames, and handles. Single videos and comma-separated video URLs can work without one.
