# Player Layouts

Easy TubeCasting currently ships with one built-in frontend display style: **Spotlight Player**.

---

## Spotlight Player

The Spotlight layout is designed for an immersive viewing experience where one video takes center stage while others stay accessible in a side rail.

- **Best For:** Showcasing a specific featured video, small playlists, or curated series.
- **Visuals:** A large, prominent video player on the top/left, and a scrollable playlist with thumbnails and titles on the bottom/right.
- **Behavior:** When a user clicks a video in the list, the main player instantly updates to that video without a page reload.
- **Single-Player Mode:** Set **Videos Per Batch** to `1` to hide the playlist rail.
- **Custom Controls:** Includes play/pause, seek, 15-second skip controls, volume, fullscreen, and captions when YouTube reports captions are available.
- **Styling:** Supports dark/light theme mode and accent color.

---

## Which one should I choose?

Use **Spotlight Player** for all current Easy TubeCasting blocks. For larger libraries, keep **Videos Per Batch** moderate so visitors can reveal more videos gradually with **Load More**.

> [!TIP] If older documentation or screenshots mention a Grid layout or grid columns, those controls are not part of the current built-in block UI.
